Welcome to Bitly Integration

Connect your Bitly account to shorten URLs directly from your project dashboard.

This integration allows you to create short, trackable links for your projects.

Where to find your Bitly access code

1. Log into your Bitly account in a new tab.

2. Go to Settings → API (or Profile → Settings → API).

3. Create or copy your personal access token.

4. Paste it below. We’ll encrypt it and store it only on this device.

Encrypting Access Code

We’re securely encrypting your Bitly access code before saving it locally.

Bitly Integration Terms

1. Data Usage

Your Bitly access code is stored locally in your browser and is never sent to our servers. You are responsible for keeping this code secure.

2. API Limits

This integration uses Bitly's API and is subject to their rate limits and terms of service. We are not responsible for any API limitations or changes made by Bitly.

3. Local Storage

Your access code is stored in your browser's local storage. If you clear your browser data, you will need to re-enter your access code.

4. Manual Installation

If you use LiteStack on multiple devices, you must manually install this integration on each device by entering your access code.

5. Liability

We are not responsible for any issues that may arise from the use of this integration, including but not limited to API errors, rate limiting, or data loss.
